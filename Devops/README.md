# Devops

This folder owns infrastructure and operational assets for BluelearnerHub.

- `aws/`: AWS deployment templates and scripts
- `database/`: SQL migrations
- `deployment/`: deployment documentation and templates
- `docker/`: container runtime assets
- `scripts/`: setup, validation, and maintenance scripts
- `tools/`: support utilities
- `docker-compose*.yml`: local and production orchestration
- `railway.json`, `render.yaml`: platform deployment descriptors

Run Docker orchestration from this directory so relative paths resolve correctly.
